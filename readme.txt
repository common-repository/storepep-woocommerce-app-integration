Sends push notification to StorePep WooCommerce App whenever changes are made on your WooCommerce Store.

== Description ==

= Introduction =
Managing your store while you are away from your computer, or when on the move is getting really easy! Now, with StorePep WooCommerce App Integration plugin, get instant alerts on your mobile about the many possible changes on your store. This plugin integrates push notification feature to the StorePep WooCommerce App, whenever the following changes are initiated on your WooCommerce Store:

<blockquote>
<ul>
<li>A <strong>New Order</strong> is placed on your store.</li>
<li>A change in <strong>Order Status</strong> (you can specify which status change should trigger in the order notification settings).</li>
<li>A <strong>New Customer</strong> is added to your store.</li>
<li>When your stock level goes below the <strong>Low Stock</strong> threshold value set in the Inventory Settings.</li>
<li>When your stock level goes below the <strong>Out-of-Stock</strong> threshold value set in the Inventory Settings.</li>
</ul>
</blockquote>

You do not have to worry anymore about the smooth functioning of your store. StorePep will promptly alert you on important matters that require your immediate attention.

= How does it work? =
StorePep WooCommerce App Integration plugin pushes the change to our Amazon EC2 server. From there, the information is forwarded to the StorePep WooCommerce App in your registered mobile when the order is placed.

== Installation ==

1. Upload the plugin folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. That's it ! you can now configure the plugin.
